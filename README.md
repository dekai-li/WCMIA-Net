# WCMIA-Net

This repository provides data availability information for the manuscript:

**WCMIA-Net: Wavelet-Enhanced Cross-Modal Interaction Attention Network for Hyperspectral-X Fusion Classification**

## Data Availability

The datasets used in this study are publicly available third-party benchmark datasets. The authors do not own these datasets and therefore do not redistribute the original data. Readers and reviewers should obtain the datasets from the original providers and comply with the corresponding terms and conditions.

The datasets used in this study are available as follows:

1. **MUUFL Gulfport hyperspectral-LiDAR dataset**  
   DOI: https://doi.org/10.5281/zenodo.1186326

2. **Berlin-Urban-Gradient dataset**  
   DOI: https://doi.org/10.2312/enmap.2016.002  
   Supplementary dataset DOI: https://doi.org/10.5880/enmap.2016.002

3. **Augsburg MDAS dataset**  
   DOI: https://doi.org/10.14459/2022mp1657312

4. **Houston2018 dataset**  
   The Houston2018 dataset is available from the 2018 IEEE GRSS Data Fusion Contest / University of Houston data access webpage. Users should follow the data provider's access instructions and terms of use.

This study did not involve human participants, human tissue, or personal data; therefore, Institutional Review Board approval was not applicable.

The source code and experimental configuration files will be released in this repository upon acceptance and are also available from the corresponding author upon reasonable request.
